# Evidence Repository (FIPS 140-2)

Place validation evidence and supporting records here. Evidence shall be dated, attributable, and protected from unauthorized modification where feasible.

- `module-spec/`
- `ports-interfaces/`
- `roles-services/`
- `state-model/`
- `physical-security/`
- `operational-environment/`
- `key-management/`
- `emi-emc/`
- `self-tests/`
- `configuration-management/`
- `other-attacks/`
