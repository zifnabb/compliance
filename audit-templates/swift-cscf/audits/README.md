# Audits (CSCF)

Store internal readiness reviews and audit artifacts here.

## Suggested contents

- Internal readiness review reports (date-stamped)
- Findings log and remediation plan
- Evidence sampling notes
- Scope statement and architecture type selection snapshot

## Suggested filenames

- `internal-audit-YYYY-MM-DD.md`
- `findings-log.md`
- `remediation-plan.md`
